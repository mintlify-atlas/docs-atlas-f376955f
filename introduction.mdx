<Note>
  MAPRISER ATOTO is for internal municipal use only. Access is restricted to staff of the H. Ayuntamiento de Atotonilco de Tula with valid institutional credentials.
</Note>

MAPRISER ATOTO is a progressive web application that lets municipal employees generate, preview, and download official administrative documents from reusable templates called **machotes**. It replaces ad-hoc document creation with a controlled, standardized workflow that applies the correct letterhead and captures only the fields each document type requires.

## Who uses this system

Any staff member of the H. Ayuntamiento de Atotonilco de Tula with an active account can use MAPRISER. The system serves three categories of users:

- **Administrators** — configure templates, manage users and areas, and access cross-departmental statistics.
- **Area heads (jefes de área)** — create and manage machotes for their assigned departments, and view their area's document history.
- **Standard users (auxiliares administrativos)** — generate documents from existing machotes and download or print the results.

## Municipal areas covered

MAPRISER covers all nine administrative areas of the Ayuntamiento:

| Area | Key |
|---|---|
| Recursos Humanos | RH |
| Secretaría | SEC |
| Tesorería | TES |
| Obras Públicas | OP |
| Desarrollo Social | DS |
| Seguridad Pública | SP |
| Catastro | CAT |
| Servicios Públicos | SERV |
| Transparencia e Informática | TEI |

Each area operates independently within the system — users can only see and generate documents for the areas they are assigned to.

## Document types

The system currently supports three document types:

- **Constancias laborales** — certificates that verify the employment relationship of a municipal worker.
- **Licencias administrativas** — authorizations for an employee to be absent from work.
- **Constancias de no adeudo predial** — certificates confirming that a property has no outstanding tax debt.

Each document type is backed by a machote that defines the letterhead and the dynamic fields a user must complete before generating the document.
